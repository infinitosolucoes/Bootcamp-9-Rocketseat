O que foi feito?

Criação de uma aplicação GymPoint, onde apenas o Usuario Administrador pode criar e alterar os dados dos alunos da academia.


Libraries utilizadas:
-Express
-Sucrase
-Yup
-Bcrypt
-Nodemon
-Sequelize
-Eslint
-Prettier
-jsonwebtoken

Softwares utilizados:
-Docker
-Postgres
-Isomnia
-Postbird
